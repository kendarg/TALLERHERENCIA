# TALLERHERENCIA
Proyecto realizado en JavaScript para practicar los conceptos de Programación Orientada a Objetos (POO), especialmente la **herencia**, creación de clases, métodos y sobrescritura de comportamientos.

## Contenido

### Punto 1 - Restaurante

Se crea una clase `restaurante` que permite almacenar información de un restaurante y verificar si tiene una buena calificación.

**Conceptos utilizados:**

* Clases
* Constructores
* Métodos
* Objetos

---

### Punto 2 - Películas VIP

Se desarrolla una clase base `peliculas` y una clase derivada `peliculaVip`, donde se modifica el cálculo del precio según los servicios adicionales incluidos.

**Conceptos utilizados:**

* Herencia
* Sobrescritura de métodos
* Uso de `super`

---

### Punto 3 - Usuarios Nequi

Se implementan las clases `usuarioNequi` y `comercioNequi` para simular envíos de dinero y manejo de comisiones.

**Conceptos utilizados:**

* Herencia
* Polimorfismo
* Reutilización de código

---

### Punto 4 - Vehículos

Se crea una jerarquía de vehículos con diferentes tipos de transporte que calculan tarifas según sus características.

**Conceptos utilizados:**

* Herencia
* Polimorfismo
* Métodos personalizados

## Tecnologías Utilizadas

* JavaScript
* Node.js
